# AI People Counter

Real-time people detection and tracking system using YOLOv8 and SORT.

## Features

- Live webcam detection
- Upload video processing
- Unique ID tracking
- Line crossing counter
- Gradio web interface

## Tech Stack

- OpenCV
- YOLOv8 (Ultralytics)
- SORT Tracker
- Gradio

## Run Locally

```bash
pip install -r requirements.txt
python app.py
